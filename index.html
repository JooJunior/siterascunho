Como surgiu

Nos anos 90, o HTML surgiu como linguagem essencial para criar páginas web. Desenvolvido por Tim Berners-Lee, um físico do centro de pesquisas CERN, na suíça,ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. A criação do HTML foi fundamental para criação e formatação de conteúdo na internet, fazendo assim, uma revolução digital e transformando a maneira como as informações são compartilhadas e acessadas.

Estrutura básica do HTML

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags , e e a instrução <!DOCTYPE>. Esta estrutura está informada em uma documentação que descreve todos os detalhes do HTML, no caso as tags e atributos, e como os navegadores devem considerar e interpretar estas tags, esta documentação é chamada de "especificação do HTML", e através do que está declarado nela que é possível entender se um documento HTML válido. Um documento HTML inválido é carregado pelo navegador, porém em um "modo de compatibilidade".

A estrutura contém basicamente html, head, body, onde esta ultima parte é onde o usuário consegue ver. Além disso, utilizamos algumas tags como de input,label, textarea,etc.

Lista ordenada e não ordenada

São utilizados para representar os itens de uma lista. As listas ordenadas são representadas por um carácter (ex:•) antes do item listado; As listas não ordenadas são representadas por números (ex:1., 2., 3.)

  • <ol> </ol>: Define uma lista ordenada
  • <ul> </ul>: Define uma lista não ordenada
  • <li> </li>: Define um item de lista
